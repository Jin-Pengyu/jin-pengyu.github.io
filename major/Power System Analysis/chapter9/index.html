    第九章：电力系统三相短路的暂态过程

    电力系统短路故障概述¶

    电力系统的稳态和暂态¶

    Q：什么是电力系统的稳态和暂态？特点是什么？

    • 电力系统的稳态：正常的、相对静止的运行状态；稳态分析——电力系统正常的、三相对称运行状态的分析和计算（如潮流计算）。稳态时节点电压和支路电流的幅值、频率、相位不随时间变化。
    • 电力系统的暂态：从一种稳定运行状态向另一种稳定运行状态的过渡过程；暂态分析——分析过渡过程中电压、电流、功率等随时间的变化（如故障分析、突增突减负荷等）。

    电力系统短路故障的原因¶

    Q：举例说明引起电力系统短路的原因有哪些？

    • 绝缘失效：有绝缘受到污染或自然老化引起。
    • 自然原因：风、雨、雪、雷、雾、鸟、兽、树。
    • 设备原因：设计、安装、维护不良等。
    • 人为原因：运行人员误操作（未拆地线合闸、带负荷拉隔离刀闸等）、放风筝、战争破坏等。

    电力系统短路故障的后果¶

    Q：举例说明引起电力系统短路的后果有哪些？

    • 产生大电流：发热，高温损坏设备，电弧将许多东西短时融化（电焊）；电磁力（F=BLI）损坏设备。
    • 造成低电压：电器设备无法正常工作，医院、矿山引起危险。
    • 严重影响供电：如大电厂母线故障导致发电机退出运行，负荷中心故障导致严重失去负荷。
    • 干扰、破坏系统的稳定运行：总出力与总负荷不平衡导致发电机加速或减速，造成系统振荡、频率变化、发电机失步、电压崩溃而造成大面积停电。

    电力系统短路故障的类型¶

    Q：电力系统有哪些故障类型，如何表示？故障概率如何？

    • 故障可能发生在发电机、变压器和线路等元件。运行经验表明：架空输电线是电力系统中比较薄弱环节，发生短路的几率最高。（发电机、变压器在电厂、变电站中，自然因素影响较小）
    • 架空输电线路中，单相接地故障概率最大（80%以上），三相故障发生的概率小，但是最为严重，对系统的安全稳定运行影响也最大。

    电力系统短路计算的目的¶

    Q：为什么要进行短路计算？

    • 电气设备选择的依据。
    • 电力系统保护配置的依据。（整定计算）
    • 网络结构规划、设计的依据。（“N-1”，切除一个元件仍然能够保持系统稳定）
    • 电力系统稳定控制措施制定的依据。
    • 电力系统短路试验是测量分析电力系统特性的重要手段。（负荷建模，动态参数，考虑暂态状态）

    无穷大电源经阻抗三相短路的理论分析¶

    无穷大电源的概念¶

    Q：什么是无穷大电源？它具有什么特点？

    • 定义：无穷大电源是指电源电压幅值和频率均为恒定的电源。
    • 特点：不论所接负荷如何变化、外电路发生何种短路故障，其端电压的大小和频率不变。
    • 因此，无穷大电源的容量为无穷大，其内阻抗为零。实际中没有真正的无穷大电源，是一个相对的概念。
    • 可以看作是由很多个同步发电机（有限大电源）并联而成，当电源的内阻抗小于短路回路总阻抗的10%时，可以认为是无穷大电源。

    【注】：无穷大电源、无限大容量电源、无限大功率电源都是一个概念。\(e(t)=\sqrt{2}Esin(wt+\varphi)(t\geq0)\)，其中\(E、w、\varphi\)均为常数。

    三相短路暂态过程的分析¶

    Q：推导无穷大电源经阻抗三相短路的短路电流表达式？

    三相短路暂态电流表达式的物理意义¶

    Q：无穷大电源经阻抗三相短路的短路电流包含哪些分量？各分量的大小与什么有关？

    • 无穷大电源外电路发生三相短路，短路暂态过程中的短路电流包含两个分量————周期分量和非周期分量。
    • 周期分量的幅值取决于电源电压和短路回路的阻抗，且在暂态过程中保持不变，它也是回路的稳态短路电流。（强制分量，非齐次特解）
    • 非周期分量是为了使电感中的电流不突变而产生的，其值在短路瞬间最大，而在暂态过程中以时间常数\(T_a\)按指数规律衰减最终为零。（自由分量，齐次通解）

    用相量图进行短路分析¶

    Q：短路电流非周期分量在什么情况下取得最大和最小值？

    • 短路前电流\(I_m\)和短路后周期电流\(I_{pm}\)相量在时间轴上的投影，代表两者的瞬时值，一般情况下两者不相等。
    • 为了保持电感电流不突变，电路中必须产生一个非周期电流，它的初值为两者之差。
    • 相量差与\(t\)轴的平行时，非周期分量的绝对值最大。显然，三相不可能同时取得最大，总有一相最大的，有一相是最小的。
    • 相量差与\(t\)轴垂直时，非周期分离为零，直接由短路前稳态到短路后的稳态。

    Q：短路电流何时能取得瞬时最大值？

    • 非周期分量取得最大值是短路电流取得瞬时最大值的前提。
    • 在短路后约半个周期内出现最大值，即短路发生后0.01秒。

    【注】：非周期分量取得最大值的条件

    1. 相量差\(I_m-I_{pm}\)有最大可能值
    2. 相量差\(I_m-I_{pm}\)在故障瞬间与时间轴平行
    3. 于是，在电感性电路中：电路原来处于空载的情况下（\(I_m=0\)），短路恰好发生在短路周期电流取得幅值的时刻，设施电源电势刚好过零。
    4. 即\(R<<x,\varphi=\varphi'=90°\)，\(\alpha=0°\)

    无穷大电源经阻抗三相短路的计算¶

    短路冲击电流¶

    Q：什么是短路冲击电流？计算式是什么？有什么用？

    • 定义：短路电流最大可能的瞬时值。
    • 条件：短路前空载、阻抗角为90°，在半个周期（0.01s）取得：\(i=I_m+I_me^{-\frac{0.01}{T_a}}=(1+e^{-\frac{0.01}{T_a}})I_m=k_{im}I_m\)
    • 短路冲击系数（1~2）是指冲击电流值对于短路交流电流周期分量幅值的倍数，发电机母线取1.9，电厂主变高压侧母线取1.85，其余地点取1.8。
    • 冲击电流用于校验电气设备和载流导体的动稳定性。

    短路电流的最大有效值¶

    Q：短路电流的最大有效值发生在哪个周期？计算式是什么？有什么用？

    • 定义：在短路过程中，任意时刻t的短路电流有效值，是指以时刻t为中心的一个周期内瞬时电流的均方根值。
      • \(I_t=\sqrt{\frac{1}{T}\int_{t-\frac{T}{2}}^{t+\frac{T}{2}} i_t^2dt}=\sqrt{\frac{1}{T}\int_{t-\frac{T}{2}}^{t+\frac{T}{2}} (i_{pt}+i_{apt})^2dt}\)
    • 条件：短路电流最大有效值出现在短路后的半个周期内（前提与冲击电流一致）：
      • \(I_{im}=\sqrt{I_p^2+[(k_{im}-1)\sqrt{2}I_p]^2}=I_p\sqrt{1+2(k_{im}-1)^2}\)（牢记）
    • 短路电流最大有效值用于校验断路器的开断能力。

    短路容量¶

    Q：什么是短路容量？计算式是什么？有什么用？

    • 定义与计算式：是短路电流最大有效值（一般取周期分量电流初始值）同短路处的正常工作电压（一般用平均额定电压）的乘积，又称为短路功率。
      • \(S_t=\sqrt{3}V_{av}I_p\)
      • \(S_{t*}=\frac{\sqrt{3}V_{av}I_p}{\sqrt{3}V_BI_B}=I_{p*}\)
    • 短路容量也主要用于校验断路器的开断能力。
    • 之所以如此定义短路容量，是因为开关要切断这么大的电流，且断流时触头应能经受住工作电压的作用。
    • 短路容量很有用，能反映电力系统某点与电源之间的紧密程度。（\(S_{t*}=I_{p*}=\frac{1}{X_\Sigma}\)）

    【注】确定冲击电流、短路电流有效值及短路功率，都必须计算短路电流的周期分量，可见对短路电流周期分量的计算是非常重要的。

    同步发电机与无穷大电源短路的差异¶

    同步发电机的绕组¶

    Q：隐极发电机和凸极发电机有什么不同？

    • 区别1：磁极不同，隐极电机的dq轴磁阻相同，凸极差异很大。
    • 区别2：凸极电机有实际的阻尼绕组，而隐极电机的阻尼绕组是实心转子的涡流作用。

    Q：同步发电机有哪些绕组？

    • 定子侧：a、b、c三相绕组ax、by、cz。
    • 转子侧：d轴上有励磁绕组ff，等效阻尼绕组DD；q轴上有等效阻尼绕组QQ。

    同步发电机三相短路分析的复杂性¶

    Q：为何对同步发电机三相短路分析十分复杂？

    • 同步发电机突然三相短路后，同无穷大电源短路一样，会出现各种周期和非周期分量。
    • 由于定、转子之间存在电磁耦合，电枢绕组中的电流会对转子电流产生影响，转子电流也会因为短路故障产生新的分量。
    • 转子电流还会继续因为电磁耦合返回去影响定子电流
    • ......

    同步发电机三相短路后的波形分析¶

    Q：从波形定性分析同步发电机三相短路后的暂态过程？

    • 短路后三相电流中均出现了非周期分量，最终衰减为零（与无穷大带能源类似），衰减时间常数为\(T_a\)。
    • 短路后三相电流中均出现了较大的周期交流分量，交流分量也会衰减，最终衰减到了某个三相一样大的稳态值。
    • 仔细看！交流分量先是再故障后迅速增大到一个值，然后迅速衰减，再慢慢慢慢地衰减到稳态。第一个衰减时间常数为\(T_d''\)，后一个时间常数为\(T_d'\)。（无穷大电源周期分量不衰减）
    • 励磁电流中出现了交流分量，比较快地衰减到零（似乎跟定子非周期分量差不多衰减一样快）；故障后直流分量会增大，然后慢慢地衰减到正常大小（跟定子周期分量衰减一样慢）

    同步发电机空载突然三相短路电流的频率分析¶

    定子电流分析¶

    • 分析前提：短路前后转子保持同步转速；转子有励磁电流；先不考虑阻尼绕组；定子回路开路（空载，负载时更复杂）。
    • 分析思路：分析磁链\(\rightarrow\)回路对应电流
    • 四个回路：定子（ax、by、cz），励磁回路（ff）

    励磁电流分析¶

    同步发电机空载突然三相短路时各分量的依存及衰减¶

    三相短路后定转子各分量的小结¶

    Q：同步发电机空载发生三相短路时定、转子将出现何种电流？这些电流分量的衰减特性如何？

    • 定子电流：出现直流分量、基频分量和倍频分量。其中：直流分量和倍频分量会衰减为零（自由分量，回路中有电阻，会损耗掉），基频分量再故障时刻最大（抵偿作用而产生的基频分量），最后衰减到一个稳态值。
    • 励磁电流：故障后出现基频分量，直流分量也会增大。其中：基频分量会衰减到零，直流分量最终会回到正常运行时的稳态值。

    定、转子各电流的依存关系¶

    Q：同步发电机三相短路后定、转子各电流如何依存？

    • 发电机空载短路瞬间，定子回路出现基频周期分量以抵消转子主磁场的交变磁链；出现直流分量及二倍频分量以维持磁链初值不变。
    • 转子回路中会因基频周期分量产生的去磁电枢反应磁场而出现增大的直流分量，会因定子直流分量和二倍频分量而产生基频分量。
    • 因此，定子的增大的基频周期分量与转子的直流分量依存；定子的直流和二倍频分量与转子的基频分量依存。

    【注】关于阻尼绕组

    • 正常运行时阻尼绕组中没有电流。（闭合线圈，无源的）
    • 故障后阻尼DD绕组中会跟励磁绕组一样出现直流和基频电流（DD和ff绕向相同）。当忽略定子电阻时，定子基频电流仅产生直轴方向的去磁电枢反应（定子内电势\(E_q\)或\(E_Q\)为q轴方向，忽略定子电阻后，回路为纯感性，只有d轴方向的电流，即只有d轴方向的去磁电枢反应），阻尼QQ绕组中没有直流分量，只有基频分量。
    • 阻尼绕组的电感小，电阻大，直流分量衰减迅速。（阻尼绕组线圈匝数少，电感因此小；阻尼条很细，电阻因此大）

    定、转子各电流的衰减¶

    Q：同步发电机三相短路后定、转子各电流如何衰减？

    • 因保持磁链不变而产生的各种无源自由分量电流都会衰减为零。
    • 各电流分量的衰减取决于直流分量电流所在绕组的时间常数，该时间常数应考虑定转子之间的耦合。
    • 定子的直流和二倍频分量与转子基频分量依存，所以三者按照定子回路的时间常数去衰减，用\(T_a\)表示。
    • 定子增大的基频周期分量与转子的直流分量依存，所以两者按照转子回路的时间常数去衰减。阻尼绕组的作用先衰减完，对应较小的时间常数为\(T_d''\)，然后仅剩励磁绕组时间常数\(T_d'\)。
